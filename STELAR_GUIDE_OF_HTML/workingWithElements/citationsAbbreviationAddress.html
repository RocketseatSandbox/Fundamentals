About the book Clean Code in JavaScript:

The book "Clean Code in JavaScript" was wrote in 2020 by the author James Padolsey.

According to the author: - Clean coding means that in the first place you write code for your later self and for your co-workers and not for the machine. (2020, Padolsey, James)
